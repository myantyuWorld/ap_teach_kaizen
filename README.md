# ap_teach_kaizen

## 概要

みんなの改善点を共起ネットワークで分類し、検索等で、類似する改善点を教えてくれる君  
※ とりあえずできてからデータはもらえばいいから、”参考”でやっているように青空文庫の適当な文書でやって、表現できるまで実装する

## 開発環境

Flask + Heroku + LineBot（あくまで暫定的なインタフェース） + 共起ネットワーク
